## project: Skill Exchange Platform for Students

It is a Java + MySQL desktop application where user add skill they can teach and skill they want to learn then this application
automatically matches students who can teach each other creating platform for students to connect to learn new skills.

## Purpose:

The Skill Exchange Platform is designed to create a collaborative learning environment where students can connect (through email) to teach and learn skills from one another. 
The system helps users share their skills in different areas such as programming, design, communication, or academics while also discovering 
new skills they want to learn. It solves the problem of limited access to peer learning opportunities by providing an organized platform for managing
user profiles, skills, and exchange requests. The application mainly targets university students who want to improve their abilities, gain practical knowledge,
and build connections through skill sharing.

## features:
1. SQL stores all data
2. users can register in this application
3. users login in their account
4. view profile
5. manage skills where you can add skills you can teach and learn
6. manage skills - delete skill from profile
7. search skills
8. view match result
9. Automatic recommendations using SQL JOIN queries
10. can see email of matched users for contact
11. any update done in application will be updated in sql automatically
